## 🌍 Live Demo
[🔗 View Live Project](https://Ashish01-starter.github.io/WEATHER-APP/)

## 📌 Description
A responsive web application that provides real-time weather information for any city using external API integration. The application displays key weather parameters such as temperature, humidity, and conditions with a clean and user-friendly interface, ensuring a smooth experience across devices.

---

## ⚙️ Tech Stack
- React
- TypeScript
- Vite
- REST API Integration
- CSS

---

## 🚀 Features
- 🌐 Search weather by city
- 📊 Real-time weather data display
- 📱 Responsive UI for all devices
- ⚡ Fast performance using Vite
- 🎯 Clean and intuitive user interface

---

```bash
# Clone the repository

# Navigate to project folder

# Install dependencies
